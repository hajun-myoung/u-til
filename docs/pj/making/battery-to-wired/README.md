## 청사진

### v1.0.0

---

**기본 아이디어**

- 배터리로 전원을 공급하는 대신, AC전원에 연결하여 동력을 공급한다.

**요구사항**

- (중요) 가정용 AC전원의 전압 220V를 제품이 요하는 정도에 맞게 바꿀 컨버터가 필요해 보인다.
- (기본) 약간의 납땜이 필요할 것이다.
- (기본) 여분의 케이블이 필요할 것이다.
