# HTML 기본 글꼴들

HTML에서 지원하는 기본 글꼴들은, 다 영어에요. 하지만, 일부 한글 글씨체의 경우 일부 디바이스에서 기본으로 지원하기도 하죠. 물론, 한글 글씨체는 될 때도 있고 안 될 때도 있어요.

이제 한글/영문 기본 글씨체들을 보기 전에, 하나만 알고 갑시다. 'serif'와 'sans serif'에 대한 내용이에요.

세리프체는 돌출된 글자, 산세리프체는 돌출되지 않은 글자에요.

세리프(serif)는 타이포그래피에서 글자와 기호를 이루는 획의 일부 끝이 돌출된 형태를 가리킨다.

세리프가 있는 글꼴은 세리프체(serif typeface, serifed typeface)라 하며, 세리프가 없는 글꼴은 산세리프체(sans-serif, 여기서 sans는 "없음"을 뜻하는 프랑스어 낱말 sans에서 비롯)로 부른다.

From. wiki 백과open in new window의 세리프 문서open in new window

더불어서, monospace는 각 글자의 여백이 똑같은 글자를 말해요. 상대적으로 글자들이 서로 떨어진 듯한 모습을 보여줘요.

# Serif 글꼴

<p style="font-family: 'Times New Roman', Times, serif;"> 내용 </p>
1

와 같이 적용 가능해요.

폰트 이름예시
Georgia, serif This is a Testing
"Palatino Linotype", "Book Antiqua", Palatino, serif This is a Testing
"Times New Roman", Times, serif This is a Testing

예문들에는 일괄적으로 30px / Bold 체가 적용되어 있어요.

# Sans-serif 글꼴

<p style="font-family: Arial, Helvetica, sans-serif;"> 내용 </p>
1

와 같이 적용 가능해요.

폰트 이름예시
Arial, Helvetica, sans-serif This is a Testing
"Arial Black", Gadget, sans-serif This is a Testing
"Comic Sans MS", cursive, sans-serif This is a Testing
Impact, Charcoal, sans-serif This is a Testing
"Lucida Sans Unicode", "Lucida Grande", sans-serif This is a Testing
Tahoma, Geneva, sans-serif This is a Testing
"Trebuchet MS", Helvetica, sans-serif This is a Testing
Verdana, Geneva, sans-serif This is a Testing

예문들에는 일괄적으로 30px / Bold 체가 적용되어 있어요.

# Monospace 글꼴

<p style="font-family: 'Courier New', Courier, monospace;"> 내용 </p>
1

와 같이 적용 가능해요.

폰트 이름예시
"Courier New", Courier, monospace This is a Testing
"Lucida Console", Monaco, monospace This is a Testing

예문들에는 일괄적으로 30px / Bold 체가 적용되어 있어요.

# 한글 글꼴

앞서 말한 것 처럼, 지금 이 페이지에 적용된 한글 글씨가 잘 보인다고 하더라도 다른 기기 / 다른 브라우저에서는 아닐 수 있으니 유의해주세요 😃
대표적으로, 일부 모바일 환경에서는 한글 글씨체들이 적용되지 않아 보일 수 있어요.

폰트 이름예시
궁서 이 문장은 예문입니다.
궁서체 이 문장은 예문입니다.
굴림 이 문장은 예문입니다.
굴림체 이 문장은 예문입니다.
바탕 이 문장은 예문입니다.
바탕체 이 문장은 예문입니다.
돋움 이 문장은 예문입니다.
돋움체 이 문장은 예문입니다.
